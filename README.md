# WEB-Raj & Raj

### We are using `YARN`

### You can also use `npm`

## `Versions`

### react: `18.2.0`

### react-types: `18.0.28`

### node-js: above `v16.18.1`

## `Commands`

1. `To install node_modules run`
```
yarn
``` 
OR 

```
npm install
```

2. `To start the project in your local you need to run` 
```
yarn start
```

OR 

```
npm run start
```

3. `lint` 
```
yarn lint
``` 

OR 

```
npm run lint
```

4. `lint fix` 

```
yarn lint:fix
``` 
OR 

```
npm run lint:fix
```

5. `prettier` 
```
yarn prettier
``` 
OR 

```
npm run prettier
```
